# Demand Forecasting & Inventory Optimisation

**Status:** In Progress

A demand forecasting and inventory optimisation system built for a mid-sized Australian grocery retailer with 54 stores and 33 product categories.

## Business Problem
- Inaccurate forecasts causing stockouts and overstock
- Inventory tying up excess working capital with no data-driven logic
- No system for deciding how much to order and when

## Tools
Python, SQL/SQLite, Prophet, statsmodels, LightGBM, Excel + Power Query, Power BI, Jupyter
